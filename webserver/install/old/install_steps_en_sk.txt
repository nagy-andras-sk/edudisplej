# EduDisplej Installation Steps / Kroky inštalácie EduDisplej

## English

This document describes the step-by-step process of the EduDisplej installation script (`install.sh`).

### 1. Privilege Check
- The script checks if it is running as root (administrator).
- If not, it prints an error and exits.

### 2. Required Packages
- Checks for required packages: zip, unzip, curl, wget.
- Installs any missing packages using `apt-get`.

### 3. Download Installation Files
- Downloads the latest installation zip from http://edudisplej.sk/install/install.zip.
- Uses a temporary directory for download and extraction.

### 4. Installation
- Backs up any existing `/opt/edudisplej` directory.
- Creates the target directory `/opt/edudisplej`.
- Extracts the downloaded zip and copies files to the target directory.

### 5. Set Permissions
- Sets directory permissions to 755.
- Sets script file permissions to 755.
- Sets config file permissions to 644.

### 6. Install Sync Script
- Copies the `edudisplej-sync.sh` script to the target directory if available.

### 7. Cleanup
- Removes temporary files and directories used during installation.

### 8. Create Systemd Service and Timer
- Creates a systemd service and timer for automatic file synchronization every 24 hours.
- Enables and starts the service and timer.

### 9. Install and Configure Apache2
- Installs Apache2 if not already installed.
- Sets up the webroot and creates a symbolic link for `index.html`.
- Restarts Apache2.

### 10. Auto-launch Browser
- Attempts to launch the default web browser to open http://localhost/index.html.

### 11. System Reboot
- Notifies the user and reboots the system after 10 seconds.

## Slovak / Slovensky

Tento dokument popisuje krok za krokom, čo robí inštalačný skript EduDisplej (`install.sh`).

### 1. Kontrola práv
- Skript kontroluje, či beží ako root (administrátor).
- Ak nie, vypíše chybu a skončí.

### 2. Potrebné balíky
- Kontroluje balíky: zip, unzip, curl, wget.
- Chýbajúce balíky nainštaluje cez `apt-get`.

### 3. Stiahnutie inštalačných súborov
- Stiahne najnovší inštalačný zip z http://edudisplej.sk/install/install.zip.
- Používa dočasný adresár na stiahnutie a rozbalenie.

### 4. Inštalácia
- Zálohuje existujúci adresár `/opt/edudisplej`.
- Vytvorí cieľový adresár `/opt/edudisplej`.
- Rozbalí zip a skopíruje súbory do cieľového adresára.

### 5. Nastavenie práv
- Nastaví práva adresárov na 755.
- Nastaví práva skriptov na 755.
- Nastaví práva konfiguračných súborov na 644.

### 6. Inštalácia synchronizačného skriptu
- Skopíruje skript `edudisplej-sync.sh` do cieľového adresára, ak je dostupný.

### 7. Upratovanie
- Odstráni dočasné súbory a adresáre použité počas inštalácie.

### 8. Vytvorenie systemd služby a časovača
- Vytvorí systemd službu a časovač na automatickú synchronizáciu súborov každých 24 hodín.
- Povolenie a spustenie služby a časovača.

### 9. Inštalácia a nastavenie Apache2
- Nainštaluje Apache2, ak ešte nie je nainštalovaný.
- Nastaví webroot a vytvorí symbolický link na `index.html`.
- Reštartuje Apache2.

### 10. Automatické spustenie prehliadača
- Pokúsi sa spustiť predvolený webový prehliadač a otvoriť http://localhost/index.html.

### 11. Reštart systému
- Upozorní používateľa a reštartuje systém po 10 sekundách.
